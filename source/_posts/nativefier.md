nativefier 就是基于 Electron 封装的，只需要一行命令就可以生成不同平台的桌面应用程序。

安装

```shell
npm install nativefier -g
```

使用

```shell
nativefier "http://medium.com"

nativefier --name "Some Awesome App" "http://medium.com"
```
